Mazkey


A ghost ctaching game . Made with my software renderer Rhine. Written in C . Also using my windowing library Darwin to replace GLFW and SDL . All Header-Only and easy to compile. All the assets are packed into a single file using PAT (also made header-only and self-made) that makes each speecific file accessible using their key name .   
[Made in July-August 2024]

Description

Catch does pesky ghosts !
in randomly generated maps .
Find the ghosts using your Ghost Activity Monitors (G.A.M) and catch them with your laser beam gun .
